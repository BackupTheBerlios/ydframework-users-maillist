Hi Francisco,

I want to use using [] for an arrayed list box or <select
name="sample[]" multiple="multiple">
by using YDFramework with this  $this->addElement( "select",
"sample[]", "", "", $a_obj ) but I found out it won't work.

However, I have found a technique on how to logically server its
purpose through this,

In my php class, its something like:
     $this->addElement( "hidden", "sample", "I want this to be in array" );

Then in my tpl script, i just use the html way of encoding:
     <select name="sample[]" multiple>
         {html_options options=$a_obj}
     </select>

Now, when I debug it for example when i hit submit from a form,
YDDebugUtil::dump ( $fields );

I have found it does work and serve its purpose to make it array.
Output is like:
array(
  'sample' =>array (
    0 => 'Waimea',
    1 => 'Koloa',
    2 => 'Kawaihau',
  ),

If you have your own technique of doing this, hope to hear it also
from you guys! Hope mine too helps!
