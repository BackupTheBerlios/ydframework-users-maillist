Hi Luis

> how can i use a class in yellow duck framework?

You only need to init the framework.
To do that, you just need to include the YDF2_init.php file.

======================
<?php

    // Init the framework
    include_once( dirname( __FILE__ ) . '/../YDFramework2/YDF2_init.php' );

    // include the yellow duck class you want using YDInclude
    YDInclude( 'YDRequest.php' );

    ...
======================


Luis, take a look to the user-guide:

HTML: http://ydframework.berlios.de/doc/index.html
PDF:  http://ydframework.berlios.de/doc/YDFramework2.pdf


.. and open the examples files.
