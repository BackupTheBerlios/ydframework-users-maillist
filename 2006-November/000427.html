> ok, but how can i use a class that i made (is in plain php,
> mycalendar.php) as 3drparty library for the YDF.


Create a new YDF addon:
	1- create a new directory inside addons, eg: ..\addons\MyCalendar\

	2- put your class file "MyCalendar.php" inside the previous directory.
	NOTE: your class must have the same name as directory!!

	3- use YDInclude( "MyCalendar.php" );
