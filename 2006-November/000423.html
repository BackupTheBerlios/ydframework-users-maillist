You could do that easily..

1- exporting the form with all extra fields

2- applying display none to extra elements (using array( "style" => 
"display:none" ) in each attribute of those extra elements.

3- Then you would only needed to create simple js function (in the 
client side) to display all extra fields and apply that function in 
checkbox attribute (eg: array( "onclick" => "toogle() )).


But... if you want to *generate* the extra elements only when the user 
clicks the checkbox.. that's another story.

If that is what you want.. take a look to my new YDAjax example 
(form_runtime).


I hope this could be your good example.
