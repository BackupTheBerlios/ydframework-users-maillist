Hi Graeme and Matthias

> I just found that you can do this by assigning the event attribute when 
> the form element is created. Such as in this example:
> 
>             // add a button to the form
>             $this->form->addElement('button', 'mybutton', 'Process the 
> Form', array( 'onclick' => 'var x = false; if(!x) { alert(1); return 
> false; }' ) );
> 
>             // add an event for 'mybutton'
>             $this->ajax->addEvent( 'mybutton', array( & $this, 
> 'processform' ), 'myform' );           
> 
> This has the effect of including javascript of the form:
> 
> 	onclick="var x = false; if(!x) { alert(1); return false; };__ydfprocessform(xajax.getFormValues('myform'));"
> 
> This is basically what you are looking for. You can see how this works 
> if you look at the last 10 lines of the code for YDAjax::addEvent. This 
> method basically checks if there is some JS already assigned to the 
> event of the form element and adds its magic afterwards.


Yes, that's right.. and, btw, you can prepend events too, eg:


$this->form->addElement('button', 'mybutton', 'Process the Form', array( 
'onclick' => '...' ) );

$this->ajax->addEvent( 'mybutton', array( & $this,'processform' ), 
'myform', null, 'prepend' );
