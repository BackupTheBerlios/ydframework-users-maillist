Hi,

I am seeing some strange behaviour when results from a
YDDatabaseObject 
are assigned to the template.

This is my code:

    $this->page->fetch();
                       
    $page_data = $this->page->getValuesAsAssocArray();
                      
    // add the data to the template
    $this->tpl->assign( 'page',      $page_data );
    $this->tpl->assign( 'page_data', $page_data );

Now you would expect that the template variables
'page' and 'page_data' 
would be set with the same data.

In this case the data in the 'page' template variable
(an associative 
array) has only two of the elements set and all of the
other fields are 
missing.
