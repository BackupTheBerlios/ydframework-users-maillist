Afaik, "Lesser GPL" (read: LGPL, the YDF licence) "permits use of the 
library in proprietary programs; using the ordinary GPL " (read: GPL) 
"for a library makes it available only for free programs"

> can I include ydframework in my own (GPL) project and distribute with the
