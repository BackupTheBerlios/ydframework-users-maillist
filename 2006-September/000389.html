Hi Graeme

> Can someone explain how the syntax addEvent( "*", ... ) is used to 
> provide YDAjax default event processing? There are no examples of this 
> so I can't see how to use it or why I would use it.

I forgot to add an example about it.. sorry.. i will :)

Look at this example:

class C{

	
	function C(){...}

	function actionDefault(){

		// create an hardcoded js function
		$this->ajax->addEvent( 'close()', array( &$this, 'doClose' ) );

		// create a default event
		$this->ajax->addEvent( '*',       array( &$this, 'dodefault' ) );
	}


	// this is the response of 'close()'
	function doClose(){
		...
		$this->ajax->returnResults();
	}


	// this is the functon that is executed for _other_ events
	function dodefault(){

		// get Javascript function and arguments
		list( $func_name, $func_args ) = func_get_args();

		// so something
		...


		$this->ajax->returnResults();
	}

}



I hope that you have understood.. if not.. just wait to see the example 
and then (with the generated code in the client) you will understand.
