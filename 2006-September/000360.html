Commited another version of YDDatabaseObject that will behave as you wish.

reset() don't have a parameter and will reset only registered field values,
select values, query values and results.
resetAll() will execute a reset() in the object and also a resetRelations()
that will reset all relation objects.

If you want to completely clear an object you can use resetObject().
