Hi db

When reseting ( with reset() or resetAll() ) i get variables reseted and 
that's is ok. the problem is that it resets _my_ variables too (an not 
only the ones used in the YDDatabaseObject).

Maybe YDDatabaseobject should reset only variables that are fields?

This way this simple code could work (as expected):

class C extends YDDatabaseobject;

	function C(){

		...
		$this->setField( 'xpto' );
	}


	function X(){

		$this->form = new YDForm( 'f' );

		$this->resetAll();

		//...
		
		$this->form->setDefaults( ... );
	}
}

As you can see, with the previous code and with the current 
YDDatabaseObject this don't work because $resetAll() of YDDatabaseobject 
deletes the form variable and it shouldn't.

What you think to make reset deleting only its fields?
