As you can see in YDF user guide ( 
http://ydframework.berlios.de/doc/ydfilesystem_using_images.html ), 
YDFSImage class only handles _local_ image files (and not web URLs).

Maybe you should use YDHttpClient or YDUrl. See ( 
http://ydframework.berlios.de/doc/ydurl.html ).
