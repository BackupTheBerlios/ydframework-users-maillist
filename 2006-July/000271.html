In a lot of cases, there is no need for two separate actions for
insert/edit. What I generally tend to do, is to include the primary key as a
hidden field in the form. If there is no primary key, an insert needs to
happen, otherwise, an edit needs to happen.

> >     function init() {

> >             myForm.submit();
> >         }
> >
> >         editButton = document.getElementById('editButton');
> >         editButton.onclick = function() {
> >             myForm.action = 'http://yourpage.html?do=edit'; // or only
> > '?do=edit' here
> >             myForm.submit();
> >         }
