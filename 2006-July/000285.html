Hi moses

>       I get two diffetrent outputs with different versions
>    
>       The following is the code to creat a text
> 
>       $this->form->addElement('text','txtFirstName','First
> Name',array("class"=>"textbox"));
> 
>       The outputs are          
> 
>       For Version := Yellow Duck Framework 2.0 (build RC1)
> 
>       the output is <div id = edit > <label
> for="form_txtFirstName">First Name</label>  <input class="textbox"
> type="text" name="form_txtFirstName" value="">   </div>
> 
> 
>       For Version := Yellow Duck Framework 2.0 (build 2.0.0-final)
> 
>       the output is <div id = edit > First Name  <input class="textbox"
> id="form_txtFirstName" type="text" name="form_txtFirstName" value="" />
> </div>
> 
> 
>       ie In the latest version there is no <label> tag for label
> Why... is there any reason for it

You must explicitly define where you want the label. In your template 
just define:

{$formName.text.label}{$formName.text.html}

this way your will get the label before the element itself.
