Moses,

the attributes array translates directly to the HTML attributes of the HTML
that is generated for the element. Any HTML attribute can be specified in
this array. You should however avoid to specify special attributes such as
"id", "name" and "value" as they are handled internally by the YDForm class.

For the options array, it depends on which form element we are talking
about. Some require options, some don't. The example "form2.php" should give
you an idea of which elements require options and which ones don't.

I will try to add this information to the user guide one day.

Cheers,


pieter
