> Hi Moses,
>
> the following code example shows you how to do it.
> This is the PHPfile:
>
> <?php
>    // Standard include
>   include_once( dirname( __FILE__ ) .
> '/../YDFramework2/YDF2_init.php');
>    // Includes
>    YDInclude('YDForm.php' );
>    YDInclude( 'YDTemplate.php');
>    // Class definition
>    classform_radiobutton extends YDRequest {
>        // Classconstructor
>        functionform_radiobutton(){
>           $this->YDRequest();
>       }
>        // Defaultaction
>        functionactionDefault() {
>            //Create theform
>           $form = new YDForm( 'form1' );
>            //Add theradiobutton
>           $form->addElement(
>               'radio', 'sex', 'Sex:', array(), array(
> 'male' => 'Male', 'female'=> 'Female')
>           );
>            //Add the submitbutton
>           $form->addElement( 'submit', 'btnSubmit',
> 'submit' );
>            //Set thedefaults
>           $form->setDefaults( array( 'sex' => 'male'
> ) );
>            //Add thetemplate
>           $tpl = new YDTemplate();
>            //Add the form to thetemplate
>           $tpl->assignForm ( 'form',$form );
>            //Display thetemplate
>           $tpl->display();
>        }
>    }
>    // Process the request
>   YDInclude( 'YDF2_process.php' );
> ?>
> The template looks as follows:
>
> {$form.tag}
>   {$form.sex.html}
>   {$form.btnSubmit.html}
> {$form.endtag}
>
> When you add aradiobutton, the 4th parameter in the
> addElement function is an arraywith the values (the
> array keys) and the names of the values (the
> arrayvalues).
>
> You also shouldn't put the radiobutton element twice
> on thetemplate as that will give strange results.
>
> Hope that makes it a little bit more clear.
>
> cheers,
>
>
> pieter
