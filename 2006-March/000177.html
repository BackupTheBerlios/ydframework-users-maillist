Hi all

What do you think of:

1- defining the html separator as a special attribute? This attribute 
would be only understand internally by checkboxgroup and not rendered in 
html. Then we could define a checkgroup like:

$form->addElement( 'checkboxgroup', 'name', 'label', array('separator' 
=> '<br />'), array( 0 => 'choice 1', 1 => 'choice 2' ) );

2- defining another special attribute for place where labels are placed. 
Currently all labels are placed on the right side but sometimes is 
useful on right.
