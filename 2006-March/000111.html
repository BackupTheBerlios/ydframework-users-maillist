The thing is the getInstance method doesn't try to connect to the database.
It just set's the YDDatabaseDriver object with the parameters.

Only at the first query or calling directly the connect() method the
connection will be made or will trigger a fatal error if it wasn't able to
connect.

If you set the error_reporting level to not show any errors, then you will
not see if the connection was unsuccessful.
