Hi Moses,

this is something on my todo list for YDFramework 3.

The problem with version 4 of PHP is that is quite difficult to build in
proper exception handling as PHP4 doesn't support throw/catch constructs.
PHP5 however does support these and will make it very straightforward to
handle situations like these.

There are some tricks we can do to emulate something like this in PHP4, but
it's very time consuming and not easy to do.
