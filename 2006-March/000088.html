That will not work indeed.

YDDatabaseObject always returns an associative array because it must fill
the object with the values and there's a $user->username and not $user->0.

But I think that you just want something like this, right?

array (
0 => 'dab',
1 => 'asdf',
)

Try getResultsAsAssocArray and forget about setting YDConfig::set(
'YD_DB_FETCHTYPE', YD_DB_FETCH_NUM );

$arr = $user->getResultsAsAssocArray( 'key', 'username' );

You must pass a key or the function will automatically try to use all
registered keys in the class.

-db
