    Hi,

I've spent the last couple of nights trying out YDajax. I'm quiteimpressed as it integrates nicely with YDF. I thought it would be niceto remove the dependency on YDform.

One improvement would be to make use of the Behaviour library fromhttp://bennolan.com/behaviour/ . This allows the resulting HTML to bequite clean since the resulting HTML does not have javascript eventattributes. This is BSD licensed.

Another toy to look at for rich AJAX affects is Rico (http://openrico.org/rico/home.page ). This is apache licensed.

The only downside to adding these things is of course bloat.
