Hi Moses,

the validate function and therefor also the form rules only get checked when
a form is submitted. It basically checks if the $_POST array is populated or
not to see if the form values are submitted or not.

Have a look at the validate function in YDForm.php. The source code is
pretty straightforward for this part.
