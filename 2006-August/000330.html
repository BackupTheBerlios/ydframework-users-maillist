Hi Matthias

> I recently tried to use multiple forms on one page with some ajax events 
> attached - unfortunately the javascript code of the first form will be 
> overwritten when calling the processEvents method of the 2nd ajax object 
> attached to the 2nd form.
> 
> Is there a way to use ajax with multiple forms on the same page? do I have to 
> use a specific order to not overwrite the events of each other form?

YDAjax was created with that feature (more than one form) in mind. The 
process should be transparent like if you are using just one form. Note 
that you should:

- create form elements with different names (even if they belong to 
different forms)

- add all object forms to YDAjax (using YDAjax::addForm( $YDFormObject ))


Anyway, i added a new YDAjax example on how to use more than one YDForm 
with YDAjax.

I hope this can solve your problem.
