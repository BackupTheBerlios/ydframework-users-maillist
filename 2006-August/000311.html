Hi Graeme

> The "button" elements always seem to be rendered at the bottom of a form 
> by the form renderer. Is there a way to place elements below "button" 
> elements? Should I be using different button-style elements - e.g. 
> "radio" elements.


Yes. You are using the YDForm html render and this render will place 
buttons at the end.

If you want a different way you can:

- (the hard choice) Create your own render

- (the simple) Create a template and add elements in a custom way to the 
template.
